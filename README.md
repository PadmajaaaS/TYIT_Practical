Subject 1: Advanced Mobile Programming

Practical 1 Title: App Bar Demonstration

Description:
This practical demonstrates the working of App Bar with menu options in Android.

 Steps to Run Practical 01:

1. Open Android Studio
2. Click on File → Open → Select Practical_01 project folder
3. Wait for Gradle to build the project
4. Connect Android device or start Emulator
5. Click Run ▶ button
6. Application will open with App Bar
7. Click menu icons to see Toast messages

Tool Used:
Android Studio
